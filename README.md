# Tennis Tracker

A small Next.js app for tracking tennis points, simple match momentum, and session stats.

## Tech Stack

- Next.js
- React
- TypeScript
- CSS Modules

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to use the tracker.

## Scripts

```bash
npm run dev
npm run lint
npm run build
```

## Current Features

- Track points for you and your opponent
- See current game status and tactical cues
- Review recent point history
- Reset the game and start a fresh session
